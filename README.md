
# 🐱 Focus Cat
<img width="1536" height="1024" alt="ChatGPT Image Apr 27, 2026, 05_04_00 AM" src="https://github.com/user-attachments/assets/9bee0a7a-a8bd-4f9e-ad3c-77519f5e4941" />

> 🧠 Минималистичное расширение для Firefox (Manifest V3), которое помогает держать фокус, ограничивать отвлечения и мягко напоминать о времени через «кот-сигнал».

## Хранилище (`browser.storage.local`)

## ✨ Возможности

* 📌 Белый список разрешённых доменов
* ⏱ Отслеживание времени вне фокуса
* 🪟 Плавающий таймер прямо на странице
* 🚨 Сигнал «кот-оverlay» при превышении лимита
* 🔘 Переключатель **Focus ON / OFF** в popup
* 💾 Сохранение настроек в `browser.storage.local`

### Фокус-контроль (старое поведение)

## 🧠 Как это работает

```text
Активная вкладка → проверка домена → сравнение с whitelist
        ↓
   разрешённый сайт? ── да → сброс таймера
        │
        нет
        ↓
  увеличение distractedTime
        ↓
  превышен лимит?
        ↓
  🐱 показывается кот-оверлей
```

## 💾 Хранилище данных

Используется `browser.storage.local`:

* `domains: string[]` — белый список
* `enabled: boolean` — включено ли расширение
* `distractedTime: number` — время отвлечения (ms)
* `lastAllowedTime: number` — последний фокус
* `limitMinutes: number` — лимит времени

## ⚙️ Логика работы

### 1. Проверка вкладки

* Расширение отслеживает активную вкладку каждые 5 секунд

### 2. Белый список

* Если домен разрешён → таймер сбрасывается

### 3. Не разрешённый сайт

* Начинается накопление времени отвлечения
* Показывается таймер на странице

### 4. Превышение лимита

* Активируется 🐱 overlay с цитатой
* Пользователь получает мягкое «напоминание вернуться в фокус»

## 🌐 Нормализация доменов

Поддерживаются разные форматы:

* `https://github.com` → `github.com`
* `github.com:443` → `github.com`
* `*.github.com` → wildcard-поддомен

✔ Поддомены поддерживаются автоматически
(`docs.github.com` → OK)

## 🚀 Установка (Firefox)

1. Открыть:

   ```
   about:debugging#/runtime/this-firefox
   ```

2. Нажать **Load Temporary Add-on**

3. Выбрать `manifest.json`

> ⚠️ После перезапуска браузера расширение нужно загрузить заново
