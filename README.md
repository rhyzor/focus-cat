# 🐱 Focus Cat

Firefox WebExtension (Manifest V3) для контроля концентрации с двумя режимами:

1. **Фокус-контроль** — прошлый функционал: считает время «вне фокуса», показывает таймер и кота при превышении лимита.
2. **Полный фокус** — новый режим: пускает только на сайты из белого списка, запускает обратный таймер сессии и автоматически выключается по завершении.

## Возможности

- Белый список доменов (включая поддомены).
- Тоггл `Фокус-контроль: ON/OFF` в popup.
- Таймер «Вне фокуса» для неразрешённых сайтов.
- «Сигнал кота» при превышении лимита отвлечения.
- Новый режим `Полный фокус: ON/OFF`:
  - блокирует все не-whitelist сайты;
  - показывает обратный таймер сессии;
  - после окончания времени показывает сообщение от кота «сеанс окончен»;
  - автоматически выключает полный фокус.

## Хранилище (`browser.storage.local`)

- `domains: string[]`
- `enabled: boolean` — режим фокус-контроля.
- `distractedTime: number`
- `lastAllowedTime: number`
- `limitMinutes: number`
- `fullFocusEnabled: boolean` — включён ли полный фокус.
- `fullFocusEndAt: number` — timestamp окончания полной фокус-сессии.
- `fullFocusDurationMinutes: number` — длительность полной сессии.

## Как это работает

### Фокус-контроль (старое поведение)

- Если активная вкладка в белом списке, `distractedTime` сбрасывается.
- Если нет — `distractedTime` растёт.
- При превышении лимита отправляется действие `cat` в `content.js`.

### Полный фокус (новое поведение)

- При запуске задаётся длительность (в минутах) и фиксируется `fullFocusEndAt`.
- Пока таймер не истёк:
  - whitelist-сайты доступны;
  - остальные сайты сразу переводятся на `blocked.html`.
- Когда таймер истекает:
  - полный фокус выключается;
  - на активной вкладке показывается сообщение от кота о завершении сессии.

## Установка локально

1. Откройте `about:debugging#/runtime/this-firefox`.
2. Нажмите **Load Temporary Add-on**.
3. Выберите `manifest.json`.
