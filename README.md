# 🐱 Focus Cat
<img width="1536" height="1024" alt="image" src="https://github.com/user-attachments/assets/dfd205cc-ba46-4ac3-ba8a-bd7a50229fb9" />


Минималистичное расширение (WebExtension Manifest V3) для Firefox, которое помогает возвращаться в фокус: отслеживает активную вкладку, считает время на сайтах вне разрешённого списка и показывает «сигнал кота» с мотивационной цитатой.

## Что делает расширение сейчас

- Ведёт **белый список доменов** (разрешённые сайты).
- Считает время «вне фокуса» на активной вкладке.
- Показывает плавающий таймер на странице, если сайт не в белом списке.
- При превышении лимита отправляет сигнал в content script и показывает overlay с котом 🐱.
- Позволяет временно выключить всю логику кнопкой **Focus: ON/OFF** в popup.
- Сохраняет настройки в `browser.storage.local`.

## Архитектура

- `manifest.json` — описание расширения, разрешения, popup, options, background и content scripts.
- `background.js` — основная логика фокуса:
  - хранение и обновление состояния;
  - расчёт времени отвлечения;
  - проверка активной вкладки каждые 5 секунд и при переключении/обновлении вкладок.
- `content.js` + `content.css` — UI внутри страниц:
  - таймер «Вне фокуса»;
  - overlay с котом и случайной цитатой из `quotes.txt`.
- `popup.html` + `popup.js` — компактный интерфейс:
  - переключатель Focus ON/OFF;
  - отображение накопленного времени отвлечения.
- `options.html` + `options.js` — настройки:
  - добавление/удаление доменов;
  - установка лимита времени в минутах.
- `blocked.html` — отдельная страница «Ты отвлёкся» (подготовлена в проекте).

## Хранилище (`browser.storage.local`)

Расширение использует ключи:

- `domains: string[]` — белый список доменов.
- `enabled: boolean` — глобальный флаг активности расширения.
- `distractedTime: number` — накопленное время вне фокуса в миллисекундах.
- `lastAllowedTime: number` — последний момент времени, когда активная вкладка была разрешённой.
- `limitMinutes: number` — лимит времени вне фокуса в минутах.

## Как работает логика

1. `background.js` получает активную вкладку.
2. Если расширение выключено (`enabled = false`) — проверка пропускается.
3. Если домен вкладки входит в белый список:
   - `distractedTime` сбрасывается;
   - отправляется команда остановки таймера в `content.js`.
4. Если домен не входит в белый список:
   - `distractedTime` увеличивается;
   - на странице запускается таймер;
   - при превышении лимита вызывается «сигнал кота» (overlay).

## Нормализация доменов

При добавлении доменов расширение нормализует ввод, чтобы удобно принимать разные форматы:

- `https://github.com/` → `github.com`
- `github.com:443` → `github.com`
- `*.github.com` → `github.com`

Проверка разрешения работает и для поддоменов (`docs.github.com` подходит под `github.com`).

## Установка локально (Firefox)

1. Откройте `about:debugging#/runtime/this-firefox`.
2. Нажмите **Load Temporary Add-on**.
3. Выберите `manifest.json` в корне проекта.

> Временное расширение нужно загружать заново после перезапуска браузера.

## Разрешения

В `manifest.json` используются:

- `tabs`
- `storage`
- `notifications`
- `host_permissions: ["<all_urls>"]`
